# hospedaje

Este Software es desarrollado con el objetivo de cumplir con la gestión de todos los procesos que se llevan dentro de un a institución que presta el servicio de alojamiento / Hospedajes a afiliados de EPS, y servicio de alimentación, transporte tanto terrestre intermunicipal como urbano y aereo y luego genera la respectiva facturación a dichas EPS.

Cueta con una versión web monilitica, desarrollada en Laravel 5.8 alojada en este repositorio.
